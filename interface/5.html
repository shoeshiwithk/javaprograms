5. Write a java proogram to create an interface called Resizable with methods resizeWidth(int width) and resizeHeight(int height) that allows an object to be resized. Create a class rectangle that implements the resizable interface and implements the resize methods. 

    interface Resizable{
        void resizeWidth(int w);
        void resizeHeight(int h);
        }
        
        class Rectangle implements Resizable{
        int oh=5;
        int ow=3;
        public void resizeWidth(int w){
        ow=w;
        System.out.println("Width is now changed to "+ow);
        }
        public void resizeHeight(int h){
        oh=h;
        System.out.println("Height is now changed to "+oh);
        }
        }
        
        class main{
        public static void main(String st[]){
        Rectangle obj= new Rectangle();
        System.out.println("Original width and height are "+obj.ow+" 
        and "+obj.oh);
        obj.resizeWidth(10);
        obj.resizeHeight(50);
        }
        }
